# Dawn
dawn取自黎明破晓之意
#### 简介：
**dawn为一个springCloud微服务框架，从零开始搭建；于个人学习使用！！！**

基于SpringBoot/SpringCloud；使用nacos作为配置与服务中心

#### 版本及环境信息：
- idea 
- JDK8 
- Springboot-2.3.0.RELEASE 
- SpringCloud-Hoxton.SR4
- Nacos 1.2.1

#### 功能计划：

- dawn-auth：认证模块
- dawn-gateway：网关模块
 > 使用springCloud-gateway中间件，完成了动态路由配置
